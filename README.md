# 看见少数人 · Seeing the Few

一个在上海发起的共创项目的招募宣言页。我们和神经多元者、身体障碍者，以及每一个曾觉得自己「不太合这个世界默认设置」的人，一起被听见，也一起决定怎样被看见。

我们相信：**障碍常常不在人的身上，而在我们共同设计出的环境里。**

**线上：** https://hannahlovegood.github.io/seeing-the-few/

## 关于这一页

这一页本身就是项目的第一件「包容性设计」展品——它遵循 WCAG 的基本原则：

- 语义化结构，屏幕阅读器可完整朗读；
- 纯键盘可操作，焦点清晰可见；
- 可调字号、可切换高对比模式（右上角）；
- 尊重系统「减少动效」设置，正文对比度均高于无障碍标准。

单文件、离线可用、无第三方依赖。报名走外部表单（金数据），填完四个问题后可获得联系方式。
