看见少数人

障碍常常不在人的身上，
而在我们共同设计出的环境里。

这是一个在上海发起的共创项目。我们和神经多元者、身体障碍者， 以及每一个曾觉得自己「不太合这个世界默认设置」的人， 一起被听见，也一起决定怎样被看见。

为什么做这件事

我们相信三件事

01

没有我们的参与，就别做关于我们的决定

被访谈、被展示的人，不是「素材」，而是共创者。我们希望当事人一步步从被讲述，走到自己主持、自己策划。

02

障碍在环境，不在人

不是轮椅使用者「上不了楼」，是这栋楼「只有楼梯」。我们不去「矫正」人，我们一起重新设计环境。

03

留下来，比热闹更重要

每一季结束，我们希望留下能被带走、被别人复用的东西——一份清单、一张地图、一本诗与口述的小册子。

怎么表达

一个故事，多个入口

不同的人用不同的感官进入世界。所以对我们来说，多模态不是为了好看，而是包容本身： 让视障者用听、听障者用看、注意力容易分散的人用短切片。同一个人的故事，我们尽量同时给出——

声音

播客与口述，最有温度，也对视障最友好。

影像

开放式字幕，尽力加上视觉描述音轨。

文字

完整逐字稿：可搜索、可读屏、可翻译。

图像

肖像与现场，每一张都配文字描述。

诗

从访谈里长出的句子，重新命名经验。

触觉与空间

展览里可触摸的展品、盲文标签、可亲身经历的动线。

现正招募

和我们一起

我们最想认识的，是当事人本人。

如果你是神经多元者、身体障碍者，或有相关的生命经验——你不是来「被记录」的， 你是来决定这个项目往哪走的。这是这个项目存在的前提。

同时，我们也在找愿意贡献一份力的伙伴。不要求你「全都会」，有一样、愿意学，就欢迎：

  • 访谈者 / 主持会倾听，能让人放松地把故事讲出来。
  • 播客 / 视频剪辑把素材做成有字幕、能被听见的成品。
  • 无障碍设计 / 前端让线上档案对每个人都可用。
  • 展览 / 空间策划把故事变成可走进的现场。
  • 诗歌工作坊带领陪大家用语言重新命名经验。
  • 文字 / 记录 / 逐字稿让每一次对话都留下痕迹。

想认识我？先花两分钟填张小表

我很想认识每一个来的人。填完下面这几个问题，你就会拿到我的微信—— 这不是门槛，是想在见面前先真诚地彼此了解一下。

表里只问四件事：

  1. 我该怎么称呼你？
  2. 你是谁？（自我认同、你的故事，自由书写）
  3. 你想要，或正在做什么？
  4. 你希望建立什么样的连结？

填完并提交后，页面就会显示我的微信，添加时备注「看见」即可。

关于这一页的无障碍设计

这一页本身，就是我们的第一件「包容性设计」展品。它遵循 WCAG 的基本原则：

  • 语义化结构，屏幕阅读器可完整朗读；
  • 纯键盘可操作，焦点清晰可见；
  • 可调字号、可切换高对比模式（右上角）；
  • 尊重系统的「减少动效」设置，正文对比度均高于无障碍标准。

如果你在使用中遇到任何不便，这本身就是我们要修的「环境里的障碍」——请告诉我们。
